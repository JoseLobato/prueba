prueba
======

Iniciando pruebas con git.

seguimos probando comandos...

$ git init 
$ git add 'archivo a subir'
$ git commit -m 'comentario'

$ git remote add origin git@github.com:JoseLobato/prueba.git
--- en caso de error 
$ git remote rm origin 

entrar de nuevo al repositorio y subir el o los archivos. 
$ git remote add origin git@github.com:JoseLobato/prueba.git
git push -u origin master
